# Structure

Defines a physical object that interacts with the electromagnetic fields.
A `structure` is a combination of a material property (`medium`)
and a `geometry`.

## Geometry

`geometry`: Defines geometric properties of the structure.

Options:

-   [Box](/03.geometry/Box.html)
-   [Sphere](/03.geometry/Sphere.html)
-   [Cylinder](/03.geometry/Cylinder.html)
-   [PolySlab](/03.geometry/PolySlab.html)
-   [GeometryGroup](/03.geometry/GeometryGroup.html)

## Medium

`medium`: Defines the electromagnetic properties of the structure\'s medium.

Options:

-   [Medium](/04.medium/01.non%20dispersive%20medium/Medium.html)
-   [AnisotropicMedium](/04.medium/01.non%20dispersive%20medium/AnisotropicMedium.html)
-   [PECMedium](/04.medium/01.non%20dispersive%20medium/PECMedium.html)
-   [Drude](/04.medium/02.dispersive%20medium/Drude.html)
-   [Lorentz](/04.medium/02.dispersive%20medium/Lorentz.html)
-   [PoleResidue](/04.medium/02.dispersive%20medium/PoleResidue.html)
-   [Sellmeier](/04.medium/02.dispersive%20medium/Sellmeier.html)
-   [Debye](/04.medium/02.dispersive%20medium/Debye.html)

## Name

`name`: Optional name for the structure.
