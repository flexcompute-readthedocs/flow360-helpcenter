# Flow360 GUI Documentation

Flow360 is a software package for solving extremely large electrodynamics problems using the finite-difference time-domain (FDTD) method.

This is the documentation for the Tidy3D Web GUI version, which is a simple way to build simulation models, submit jobs, and analyze results of completed runs.
